Documentation

Provides users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information and create a list of their favorite movies.

User Requests

Business Logic URL HTTP Method Request body data format Response body data format
Add user /users POST JSON object holding data about the user to add, structured like:
{
name: "Jack Bauer“,
username: "actionLover“,
password: "badpw123“,
email: "actionLover@yahoo.com"
} 		JSON object holding data about the user that was added, structured like:
{
ID: "4ef03c6ac21858b16a8ba0dc", name: "Jack Bauer“,
username: "actionLover“,
password: "badpw123“,
email: "actionLover@yahoo.com",
favorites: "null" }
Update User Info users/:username/:infoToUpdate PUT None A text message indicating if the username was updated.
Remove User by Username /users/:username DELETE None A text message indicating whether the user was removed.
Add Movie to List of Favorites /users/:username/favorites/:movieID POST None A text message indicating whether the movie was added to the favorites list and the updated list of favorite movies.
Remove Movie from List of Favorites /users/:username/favorites/:movieID DELETE none A text message indicating whether the movie was removed from the list and the updated list of favorite movies.

Movie Requests

Business Logic URL HTTP Method Request body data format Response body data format
Get List of all Movies /movies GET None JSON object holding data about all movies.
Get Data about Specific Movie by Title /movies/:title GET None JSON object holding data about a single movie. Example:
{
title: "The social Network“,
director: {
name: "David Fincher“,
born: "1962“,

Get Data about Genre by Name /movies/genres/:name GET None JSON object holding data about a movie genre. Example:
{
id: "1", name: "biopic",
description: "A film that dramatizes the life of a non-fictional person. Attempts to comprehensively tell a
single person's life story or at least the most historically important years of their lives."
